00. Introducao
Das funcionalidades gratuitas ao plano Pro

Nos modulos anteriores, voces aprenderam a usar o Claude com prompts estruturados, Projects e Custom Instructions — tudo disponivel no plano gratuito. Agora vamos explorar as funcionalidades que so existem no plano Pro e que transformam o Claude de um assistente de texto em uma plataforma de produtividade completa.

Cada feature sera apresentada no formato Teoria + Demonstracao + Atividade. O objetivo nao e que voces dominem cada ferramenta hoje — e que entendam o que existe, vejam funcionando e consigam avaliar o retorno para o proprio departamento.

Importante: As features a seguir sao do plano Pro ($20/mes ou $240/ano). Voces nao precisam assinar hoje — o objetivo e conhecer o que existe e avaliar se faz sentido para o seu trabalho. As atividades praticas foram desenhadas para funcionar tambem no plano gratuito, simulando o raciocinio de uso.

Objetivos de Aprendizagem

  1. Conhecer as 5 features exclusivas do Claude Pro: Deep Research, Connectors, Skills, Cowork e Claude Code
  2. Ver cada feature funcionando ao vivo em um caso de uso real da Bram Offshore
  3. Praticar o raciocinio de aplicacao: identificar onde cada feature geraria valor no proprio departamento
  4. Construir um Mapa de ROI consolidado com as 5 features para apresentar a lideranca
Feature 01 — Deep Research
01. Deep Research — Teoria
Pesquisa autonoma com relatorio completo

Deep Research e a funcionalidade que transforma o Claude em um pesquisador autonomo. Em vez de responder com base apenas no que ele ja sabe, o Claude navega pela internet em tempo real, cruza dezenas de fontes, avalia a qualidade das informacoes e entrega um relatorio estruturado com citacoes.

A diferenca fundamental: enquanto uma busca no Google retorna uma lista de links que voce precisa ler um por um, o Deep Research le tudo por voce, sintetiza as informacoes relevantes e entrega um documento pronto para uso — com as fontes referenciadas para verificacao.

Investigacao autonoma

O Claude navega pela web sozinho, abrindo e lendo paginas, artigos, documentos e relatorios. Ele nao depende de voce fornecer links — ele encontra as fontes relevantes por conta propria.

Tempo de processamento

Uma pesquisa com Deep Research leva entre 2 e 15 minutos, dependendo da complexidade. O Claude esta de fato navegando, lendo e cruzando informacoes — nao e uma resposta instantanea.

Relatorio com citacoes

O resultado e um relatorio estruturado com todas as fontes citadas. Voce pode verificar cada informacao clicando na referencia — nada e inventado.

Integracao Google Workspace

O relatorio pode ser exportado diretamente para o Google Docs, facilitando o compartilhamento e a edicao colaborativa com a equipe.

Feature 02 — Connectors
04. Connectors — Teoria
O Claude conectado as suas ferramentas de trabalho

Connectors sao integracoes que permitem ao Claude acessar diretamente as ferramentas que voce ja usa no trabalho — Google Drive, Gmail, Google Calendar, Slack e mais de 50 outros servicos. Em vez de copiar e colar informacoes entre sistemas, o Claude busca, le e age sobre os dados diretamente na fonte.

A tecnologia por tras e o MCP (Model Context Protocol) — um padrao aberto criado pela Anthropic que permite ao Claude se conectar a qualquer ferramenta externa de forma segura. Cada Connector funciona como uma ponte bidirecional: o Claude pode ler informacoes do servico e, em alguns casos, executar acoes nele.

Google Drive

O Claude busca e le documentos diretamente no seu Drive. Voce pode pedir "resuma o relatorio de operacoes da semana passada" e ele encontra o arquivo, le e resume — sem voce precisar abrir o documento.

Gmail

O Claude acessa seus e-mails para buscar informacoes, encontrar conversas anteriores e redigir respostas. Voce pode pedir "encontre o ultimo e-mail do fornecedor sobre a entrega de EPIs" e ele localiza.

Google Calendar

O Claude ve sua agenda e pode ajudar a organizar compromissos. Voce pode pedir "quais reunioes tenho amanha" ou "sugira um horario para a reuniao de seguranca da semana que vem".

Slack

O Claude acessa canais e mensagens do Slack para buscar informacoes discutidas pela equipe. Voce pode pedir "o que foi discutido no canal de operacoes sobre a embarcacao X".

50+ ferramentas

Alem das integracoes nativas, o ecossistema MCP permite conectar Notion, Jira, GitHub, Asana, Trello, Salesforce e dezenas de outros servicos — com novas integracoes sendo lancadas toda semana.

Seguranca e Governanca

Cada Connector exige autorizacao explicita do usuario. O Claude so acessa o que voce permite e so ve os dados que voce autoriza. As conexoes podem ser revogadas a qualquer momento nas configuracoes do Claude.

Nota sobre governanca na Bram

Antes de ativar Connectors em escala, a equipe de TI deve avaliar quais integracoes sao compativeis com as politicas de seguranca da informacao da Bram Offshore. O uso com dados sensiveis requer alinhamento previo com a gestao.

Caso de uso — Bram Offshore

Feature 03 — Skills
07. Skills — Teoria
Automatizando tarefas com instrucoes reutilizaveis

Skills sao instrucoes reutilizaveis que ensinam ao Claude como executar uma tarefa especifica de forma padronizada. Em vez de reescrever o mesmo prompt toda vez que voce precisa de um tipo de documento, voce cria uma Skill uma unica vez e o Claude aplica automaticamente sempre que detectar uma situacao relevante.

Pense em Skills como "receitas" que o Claude memoriza. Uma Skill pode definir como gerar um relatorio de inspecao, como redigir um e-mail de follow-up para fornecedores ou como analisar dados de manutencao. Existem Skills pre-construidas pela Anthropic e voce pode criar as suas proprias.

Ativacao automatica

O Claude detecta quando uma Skill e relevante para o que voce esta pedindo e a ativa automaticamente. Voce nao precisa lembrar de chamar a Skill — ela funciona no fundo.

Sem codigo

Criar uma Skill e tao simples quanto escrever instrucoes em linguagem natural. Nao e necessario programar — basta descrever o que o Claude deve fazer, em que formato e com quais regras.

Resultado consistente

Uma vez definida, a Skill garante que o resultado sempre siga o mesmo padrao. Todos os relatorios saem no mesmo formato, todos os e-mails seguem o mesmo tom — sem variacao.

Todos os contextos

Skills funcionam em qualquer conversa, dentro ou fora de Projects. Uma vez criada, ela esta disponivel em toda a sua conta Claude.

Feature 04 — Cowork
10. Cowork — Teoria
O Claude trabalhando no seu computador

Cowork e a funcionalidade que permite ao Claude acessar e trabalhar com arquivos diretamente no seu computador. Em vez de fazer upload de um arquivo por vez no chat, voce aponta o Claude para uma pasta e ele le, analisa e processa todos os arquivos de uma vez — planilhas, documentos, PDFs, imagens.

A diferenca fundamental: enquanto o Claude no chat trabalha com um arquivo por vez e perde o contexto entre conversas, o Cowork opera no nivel do sistema operacional. Ele pode abrir, ler e cruzar informacoes de multiplos arquivos simultaneamente, gerar novos documentos e salvar diretamente no seu computador.

Acesso a arquivos locais

O Claude le arquivos diretamente do seu computador — planilhas Excel, PDFs, documentos Word, CSVs. Nao e necessario fazer upload: ele acessa a pasta que voce indicar.

Execucao multi-step

O Claude pode executar tarefas complexas com multiplas etapas: ler 5 planilhas, consolidar os dados, gerar um relatorio e salvar o resultado — tudo em uma unica instrucao.

Integracao desktop

Funciona como um aplicativo nativo no Mac e Windows. O Claude ve o que voce ve na tela e pode interagir com as ferramentas que voce usa no dia a dia.

Segundo plano

Tarefas longas podem rodar em segundo plano enquanto voce continua trabalhando. O Claude avisa quando terminar.

Caso de uso — Bram Offshore

Daniel, Manutencao

Daniel recebe 4 planilhas de manutencao de diferentes embarcacoes toda semana. Hoje, ele abre cada planilha, copia os dados relevantes, cola em uma planilha consolidada e formata manualmente. Com Cowork, ele aponta o Claude para a pasta com as 4 planilhas e pede: "Consolide os dados de manutencao pendente de todas as embarcacoes, organize por prioridade e gere um relatorio resumido." O Claude faz em 2 minutos o que Daniel leva 90 minutos para fazer.

Pergunta para o grupo

Qual tarefa do dia a dia de voces envolve abrir multiplos arquivos, copiar dados entre eles e consolidar manualmente? Esse e o caso de uso perfeito para o Cowork.

Feature 05 — Claude Code
13. Claude Code — Teoria
Programacao e automacao sem ser programador

Claude Code e a funcionalidade que permite ao Claude escrever, executar e depurar codigo diretamente. Isso nao significa que voces precisam virar programadores — significa que voces podem pedir ao Claude para criar ferramentas, scripts e automacoes que resolvem problemas reais do trabalho, sem escrever uma linha de codigo.

O potencial para trainees: tarefas repetitivas que envolvem processamento de dados, geracao de relatorios, analise de planilhas e criacao de ferramentas internas simples podem ser automatizadas com um unico prompt. O Claude escreve o codigo, executa e entrega o resultado.

Analise de dados

O Claude pode escrever scripts para analisar grandes volumes de dados, gerar graficos, identificar padroes e criar dashboards — tudo a partir de uma descricao em linguagem natural do que voce precisa.

Automacao

Scripts que automatizam tarefas repetitivas: renomear centenas de arquivos, converter formatos, extrair dados de PDFs, consolidar planilhas, gerar relatorios periodicos.

Ferramentas simples

Calculadoras internas, formularios de registro, checklists interativas, dashboards de acompanhamento — ferramentas que o departamento precisa mas que ninguem teve tempo de criar.

Trainees Tech

Para trainees de TI, Claude Code e uma ferramenta de produtividade direta: debugging, code review, documentacao tecnica e prototipagem rapida de solucoes.

Caso de uso — Bram Offshore

Felipe precisa criar um script que leia uma planilha de controle de horas de bordo, identifique tripulantes que excederam o limite de horas semanais e gere um alerta automatico. O Claude escreve, testa e entrega o script funcionando.
