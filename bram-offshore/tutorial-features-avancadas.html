01. Deep Research — Pesquisa Profunda

O Claude pesquisa na internet, consolida multiplas fontes e entrega um relatorio estruturado em minutos. Util quando voce precisa de uma visao abrangente, nao apenas uma resposta rapida.

PROMPT DE DEMO — INSTRUTOR COPIA Pesquise as melhores praticas de gestao de incidentes HSE em operacoes offshore no Brasil. Inclua: - Regulamentacoes vigentes (Marinha, ANP, ISGOTT) - Benchmarks de empresas do setor - Tendencias de uso de tecnologia para prevencao - Recomendacoes praticas para equipes de bordo Formate como relatorio executivo com sumario, secoes e referencias.

Casos de Uso por Area

Linha de Frente Normas de seguranca antes de uma operacao nova
Corporativo Benchmark de mercado para apresentar ao gestor
Tech Levantamento de solucoes para um problema tecnico

02. Connectors — Integracoes

03. Skills — Tarefas Padronizadas

Uma Skill e um conjunto de instrucoes salvas. Voce escreve uma vez — o Claude reproduz o padrao toda vez, sem variacoes.

Regra pratica

Se voce usou o mesmo prompt mais de duas vezes, ele deveria ser uma Skill.

DEMO — INSTRUTOR CRIA SKILL AO VIVO Nome: Relatorio de Ocorrencia HSE Gatilho: Use quando eu descrever os detalhes de uma ocorrencia de seguranca ou incidente offshore. Instrucoes: Gere um Artifact de relatorio de ocorrencia com: 1. Dados do incidente (data, localizacao, tipo) 2. Descricao cronologica do ocorrido 3. Analise de causa raiz — metodologia 5 Porques 4. Acoes corretivas imediatas 5. Acoes preventivas de longo prazo 6. Campo para assinatura do responsavel e data Formate em padrao profissional pronto para registro e auditoria.

Casos de Uso por Area

HSE Padronizar relatorios de ocorrencia
Financeiro Analises mensais com mesma estrutura
RH Comunicacoes de onboarding padronizadas
Comercial Propostas com estrutura consistente

05. Claude Code — Desenvolvimento

Um ambiente onde o Claude escreve, testa e executa codigo no seu computador. Para quando a tarefa vai alem de um Artifact simples.

PROMPT DE DEMO — INSTRUTOR COPIA Crie um script Python que: 1. Leia uma lista de embarcacoes com horas de bordo mensais 2. Calcule a media de horas por mes para cada embarcacao 3. Identifique quais ficaram abaixo da meta de 300h/mes 4. Gere um relatorio em texto com o resultado formatado Use dados ficticios: 3 embarcacoes de apoio offshore, dados de Jan a Mar.
Perspectiva

O que voces construiram no Lovable em horas, com Claude Code + uma boa especificacao, poderia ter saido em minutos.

Casos de Uso por Area

Tech Ferramentas internas, analise de dados, automacoes
Financeiro Modelagem e analise de dados

06. Artefato Final — Proposta + Top 3

Voces preencheram 5 blocos de mapeamento. Agora vamos juntar tudo em um unico documento.

1
Abram o Claude, dentro do Project criado no Modulo 03.
2
Colem o prompt abaixo e substituam o campo indicado pelos 5 blocos de mapeamento preenchidos durante a aula.
3
Rodem o prompt e revisem o Artifact gerado.
PROMPT FINAL — VOCES FAZEM AGORA (DENTRO DO PROJECT) Ao longo da aula de hoje mapeei 5 oportunidades de uso do Claude Pro no meu departamento — uma para cada feature apresentada. Aqui estao: [cole os 5 blocos de mapeamento que voce preencheu durante a aula] Com base nesses dados, gere um Artifact em HTML com dois blocos: BLOCO 1 — PROPOSTA PARA A LIDERANCA Titulo: "Por que o Claude Pro faz sentido para o meu departamento" - Tabela com as 5 oportunidades: tarefa atual | tempo hoje | resultado com Claude | ganho estimado - Paragrafo de conclusao com o total de horas economizadas por mes - Custo: $20/mes por assinatura ($240/ano) - Frase final: "O investimento se paga com [X horas economizadas] ainda no primeiro mes." BLOCO 2 — TOP 3 PARA A FASE 2 - As 3 oportunidades de maior impacto, ordenadas por potencial - Para cada uma: o que seria implementado, qual feature usaria e qual seria o resultado esperado Identidade visual BRAM OFFSHORE (aplicar em todo o documento): - Header principal: fundo azul marinho (#003087), texto branco, "BRAM OFFSHORE" em Montserrat Bold caixa alta, departamento abaixo - Fundo do documento: branco (#FFFFFF) - Titulos dos blocos: azul marinho (#003087), Montserrat Bold - Header das tabelas: azul marinho (#003087), texto branco - Linhas alternadas das tabelas: branco e azul claro (#E8F0F8) - Numeros de ganho e metricas de impacto: ambar (#F5A800) em negrito - Texto do corpo: cinza escuro (#333333), Open Sans - Secao de custo/ROI: fundo azul marinho (#003087), texto branco, valor de ROI em ambar (#F5A800) em destaque - Rodape: fundo azul marinho (#003087), texto branco - Estilo: corporativo, limpo, profissional — pronto para apresentar a gestora de RH
