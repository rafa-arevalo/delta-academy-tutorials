01. Prompt fraco vs. prompt forte

Mesmo pedido, resultados completamente diferentes.

Prompt Fraco

"Me ajuda com uma checklist de inspecao de embarcacao."
Generico, sem estrutura, sem papel definido, sem formato de saida. A IA vai chutar.

Prompt Forte

"Voce e tecnico de seguranca offshore. Crie uma checklist de inspecao pre-operacional para embarcacao de apoio maritimo. Cubra: navegacao, salvatagem, HSE, motores e documentacao regulatoria. Formate com checkboxes, campo para numero da embarcacao, data e assinatura do responsavel."
Papel definido, contexto claro, tarefa especifica, formato de saida detalhado.

Os 4 Elementos de um Prompt Forte

#ElementoExemplo
1Papel"Voce e tecnico de seguranca offshore..."
2Contexto"...embarcacao de apoio maritimo..."
3Tarefa clara"...checklist com estas secoes..."
4Formato de saida"...com checkboxes, campos e assinatura."

02. O que e um Artifact

A diferenca entre uma resposta comum e um documento pronto para uso.

Resposta normalArtifact
ApresentacaoTexto corrido no chatJanela lateral com conteudo renderizado
UsoVoce copia e reformataVoce clica e usa direto
AplicacaoPara leituraPara documento, tabela, grafico ou codigo
Como ativar

Especifique o formato no prompt: "Crie um documento com..." / "Gere uma tabela em HTML..." / "Escreva o codigo Python para..."

04. Demo 2 — Prompt forte + Artifact ao vivo

Agora o instrutor envia o prompt completo. Observem a diferenca.

PROMPT — INSTRUTOR COPIA Voce e tecnico de seguranca offshore certificado pela Marinha do Brasil. Crie uma checklist de inspecao pre-operacional para embarcacao de apoio maritimo, no formato de documento profissional como Artifact em HTML. A checklist deve cobrir: - Sistemas de navegacao e comunicacao - Equipamentos de salvatagem (balsas, coletes, macacoes) - Sistemas HSE (extintores, detectores de gas, EPIs disponiveis) - Motores principais e auxiliares (nivel de oleo, temperatura, alarmes) - Documentacao regulatoria (certificados validos, NORMAM-01) Inclua: checkboxes, campo para numero da embarcacao, data e assinatura do responsavel tecnico. Pronto para uso operacional. Identidade visual BRAM OFFSHORE: - Header com fundo azul marinho (#003087), texto branco, nome "BRAM OFFSHORE" em Montserrat Bold caixa alta - Fundo do documento: branco (#FFFFFF) - Titulos das secoes: azul marinho (#003087), Montserrat SemiBold - Texto do corpo: cinza escuro (#333333), Open Sans - Linha separadora entre secoes: azul claro (#E8F0F8) - Campos de assinatura e data com borda sutil em cinza - Rodape com fundo azul marinho (#003087) e texto branco
Resultado

O Artifact esta pronto para imprimir ou enviar. Mesmo pedido, resultado completamente diferente.

05. Pratica

Agora voces fazem. Adaptem o prompt abaixo para uma tarefa real do seu departamento.

PROMPT GUIADO — VOCES FAZEM AGORA Voce e um profissional de [escolha: manutencao offshore / recursos humanos / financeiro / supply chain / operacoes / TI / HSE / juridico / comercial]. Gere um Artifact em HTML no formato de documento profissional para a seguinte tarefa do meu departamento na Bram Offshore: [descreva uma tarefa concreta — ex: "elaborar checklist de onboarding" / "redigir e-mail de follow-up para fornecedor" / "criar relatorio de ocorrencia HSE" / "gerar resumo executivo de contratos"] O documento deve estar pronto para uso real, com todos os campos necessarios. Identidade visual BRAM OFFSHORE: - Header com fundo azul marinho (#003087), texto branco, nome "BRAM OFFSHORE" em Montserrat Bold caixa alta, departamento abaixo - Fundo do documento: branco (#FFFFFF) - Titulos das secoes: azul marinho (#003087), Montserrat SemiBold - Texto do corpo: cinza escuro (#333333), Open Sans - Destaques e dados relevantes: ambar (#F5A800) - Rodape com fundo azul marinho (#003087) e texto branco - Estilo corporativo, limpo e profissional
