Introdução
01. Objetivo de aprendizagem
Dar à AI os olhos da sua marca

Criar a identidade visual da sua empresa usando o Claude Design e salvá-la dentro do seu Project — para que todos os materiais que você gerar nas próximas atividades saiam com as cores, fontes e estilo da sua empresa.

Introdução
02. Conceito — Claude Design
Identidade visual sem precisar saber design

Toda empresa tem uma identidade visual: cores, fontes, estilo de comunicação. Quando você pede para o Claude criar uma apresentação ou um dashboard sem passar esse contexto, ele inventa uma estética genérica que não representa ninguém.

O Claude Design resolve isso. Você envia prints do site da sua empresa e o Claude analisa as imagens, identifica as cores, a tipografia e o estilo visual, e monta um Design System completo — sem você precisar saber o nome técnico de nada.

A virada

Depois você salva essa identidade dentro do seu Project. A partir dali, toda vez que você pedir ao Claude para criar um artefato, uma apresentação ou um dashboard, ele já sabe como a sua empresa se parece.

Demonstração
03. O que você vai ver
Design System de uma empresa de logística criado ao vivo

Criação do Design System de uma empresa de logística no Claude Design usando prints do site. O Claude analisa as imagens, extrai paleta de cores com códigos hex, tipografia e padrões visuais, e apresenta o resultado para revisão.

Em seguida, as informações extraídas são salvas no Project da aula como documento de identidade visual.

Demonstração
04. O que observar
4 detalhes do Claude Design na prática
  • Códigos hex exatos. O Claude extrai os códigos de cor exatos diretamente dos prints — você não precisa saber os números.
  • Organização em seções. O resultado vem organizado em Brand Colors, Type, UI Kit — com botões "Looks good" e "Needs work..." para revisar cada item.
  • Honestidade sobre limitações. Se não conseguiu extrair a fonte exata, o Claude informa e sugere alternativas próximas.
  • Chat no painel esquerdo. Você pede ajustes em linguagem natural diretamente ali, sem refazer o upload.
Atividade Prática
05. Antes de começar — prepare os prints do site
2 a 3 prints do site da sua empresa

Antes de acessar o Claude Design, tire 2 a 3 prints do site da sua empresa no celular ou computador:

Prints necessários

  • Print 1 — Topo da página inicial. Onde fica o logo e o menu principal.
  • Print 2 — Primeira seção de conteúdo. A área principal com cores e textos.
  • Print 3 — Qualquer página interna (opcional). Sobre, serviços ou produtos.
⚠ Atenção ao nome dos arquivos

O Claude Design não aceita arquivos com acentos, espaços ou caracteres especiais no nome. Antes do upload, renomeie para nomes simples como print01.png, print02.png, print03.png. No Mac, prints automáticos vêm com nomes tipo "Captura de Tela 2026-04-22 às 19.59.28.png" — renomeie antes de subir.

Atividade Prática
06. Passo a passo
5 passos + 2 prompts (um para rodar no Project enquanto processa, outro para extrair o Design System)
1
Acesse o Claude Design. (30 seg)
Acesse claude.ai/design. Na parte superior da tela, clique na aba "Design systems".
2
Crie o Design System. (1 min)
Clique no botão "Create" ao lado de "Create new design system — Teach Claude your brand and product". Faça upload dos prints em "Add fonts, logos and assets" e clique em "Continue to generation". Enquanto processa (~5 min), rode o Prompt A abaixo no seu Project para aproveitar o tempo.
3
Revise o Design System gerado. (1 min)
Quando terminar, o Claude apresenta o resultado em dois painéis — Chat (esquerda) e Design System (direita). Leia as limitações no chat e corrija em linguagem natural se algo estiver errado (ex.: "A cor principal da nossa marca é verde, não azul."). No painel direito, marque "Looks good" ou "Needs work..." em cada item.
4
Publique o Design System. (15 seg)
No topo do painel direito, localize a mensagem "Your design system is ready, but your feedback will improve it!". Logo abaixo há um checkbox "Published" — clique para marcar (fica azul). Isso ativa o Design System para todos os projetos futuros no Claude Design. Em seguida, rode o Prompt B abaixo no chat do Claude Design para extrair o Design System em texto.
5
Salve no seu Project. (1 min)
Abra o seu Project (AP03). Em "Adicionar conteúdo", preencha:
• Título: Identidade Visual — [nome da empresa]
• Conteúdo: cole o texto extraído no Passo 4.
Clique em "Adicionar conteúdo" para salvar.

Prompt A — para rodar no Project enquanto o Design System processa
Com base nos documentos deste Project — meu contexto profissional, as oportunidades de AI mapeadas na Matrix Kai-Fu Lee e o que as empresas do meu setor já estão fazendo com AI — mapeie as 3 situações concretas do meu trabalho na próxima semana onde eu posso aplicar o Claude. Para cada situação: - Qual funcionalidade do Claude usar: Pesquisa Profunda, Project, Artefato, Skill, Cowork ou Claude Code - O que fazer com o Claude nesta situação (1 frase de ação) - Qual resultado o Claude vai me entregar Seja direto. Máximo 1 linha por item.
Prompt B — para extrair o Design System (use no chat do Claude Design)
Resuma o Design System que você criou em formato de texto estruturado para eu salvar como documento de referência. Inclua: - Nome das cores principais com código hex - Fontes identificadas (ou substitutas sugeridas) - Tom de voz e personalidade da marca - Estilo visual em 3 adjetivos Formato: texto corrido organizado em tópicos, sem código, sem markdown técnico. Máximo 1 página.
Detalhe do Claude Design

Não existe botão de exportar — o Prompt B é o jeito de transformar o Design System em texto copiável para o seu Project.
