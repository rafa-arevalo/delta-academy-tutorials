Introdução
01. Objetivo de aprendizagem
Um dashboard gerencial do seu departamento, pronto para abrir no navegador

Usar o Claude Code para criar um dashboard gerencial do seu departamento — um painel visual interativo que exibe os dados de OKRs em gráficos e cards, com a identidade visual da sua empresa, que abre direto no navegador.

Introdução
02. Conceito — Claude Code
Criar aplicações descrevendo o que você quer

O Claude Code transforma descrições em linguagem natural em aplicações funcionais. Você não escreve código — você descreve o que quer ver, e o Claude Code constrói o arquivo para você.

A diferença para o chat e para o Cowork é o tipo de entregável: o chat entrega texto, o Cowork entrega documentos e apresentações, o Claude Code entrega aplicações que rodam no seu computador — dashboards, ferramentas, calculadoras, sistemas de gestão.

O dashboard desta atividade

Puxa os dados de OKRs da AP06, aplica a identidade visual da AP04 e entrega um painel executivo que abre no navegador como se fosse um site — mas fica salvo na sua pasta e funciona sem internet.

⚠ Pré-requisito

O Claude Desktop precisa estar instalado e você precisa ter o Git instalado no seu computador. Veja as instruções na próxima seção antes de continuar.

Antes de começar
03. Instalação do Git
Pré-requisito silencioso do Claude Code — 3 a 4 cliques

O Git é um programa gratuito que o Claude Code precisa para trabalhar com arquivos no seu computador. Pense nele como um assistente de organização que o Claude Code usa nos bastidores — você não vai precisar usá-lo diretamente em nenhum momento.

Primeiro verifique se já tem o Git instalado

A forma mais simples é perguntar ao próprio Claude Desktop. Abra o Claude Desktop, vá para a aba Code, selecione a pasta Delta Board e digite o prompt abaixo. Se o Claude Code responder que sim, pode pular esta seção.

Prompt de verificação
Você consegue verificar se o Git está instalado no meu computador?

No Mac — instalação em 3 cliques

  • Acesse git-scm.com/download/mac no seu navegador
  • Clique no botão de download — o arquivo .pkg vai ser baixado na pasta Downloads
  • Abra o arquivo baixado, clique em Continuar nas telas e clique em Instalar

Pronto. Você não vai precisar abrir nem usar o Git diretamente — o Claude Code faz isso nos bastidores.

No Windows — instalação em 4 cliques

  • Acesse git-scm.com/download/win no seu navegador
  • O download começa automaticamente — aguarde o arquivo .exe
  • Abra o arquivo e clique em Next em todas as telas (as configurações padrão funcionam perfeitamente)
  • Na última tela, clique em Finish

Pronto. Feche e reabra o Claude Desktop para ele reconhecer a instalação.

04. O que você vai ver
Dashboard gerencial de logística construído ao vivo

O professor abre o Claude Desktop, clica na aba Code, seleciona a pasta Delta Board — que já tem os arquivos de OKRs e de identidade visual da empresa de logística — e descreve o dashboard que quer criar.

Demonstração
05. O que observar
4 características do Claude Code na prática
  • Leitura automática de arquivos. O Claude Code lê os arquivos da pasta automaticamente — você não precisa copiar nem colar nada.
  • Conversação em português. Você descreve o que quer em português, como faria com um designer ou analista.
  • Dashboard real no navegador. O arquivo gerado abre no navegador como um dashboard real — com gráficos, cores e dados do departamento.
  • Iteração em linguagem natural. Se algum elemento visual não estiver certo, você pede a alteração em linguagem natural e o Claude Code atualiza o arquivo na hora.
Demonstração
06. Prompt da demonstração
Contexto: Operações · empresa de logística · Q1 2026
Prompt da demo — Dashboard de OKRs
Leia os arquivos "okrs-logistica.csv" e "identidade-visual.txt" desta pasta e crie um dashboard gerencial em HTML para o departamento de Operações de uma empresa de logística. O dashboard deve ter: CABEÇALHO: - Nome do departamento: Operações - Período: Q1 2026 - Aplicar as cores e fontes do arquivo identidade-visual.txt CARDS DE RESUMO (linha no topo): - Total de Key Results: número total - No prazo: quantidade e percentual em verde - Em risco: quantidade e percentual em amarelo - Atrasado: quantidade e percentual em vermelho GRÁFICO DE BARRAS: - Um gráfico mostrando o % de atingimento de cada Key Result, com a cor da barra indicando o status (verde/amarelo/vermelho) TABELA DETALHADA: - Todas as colunas da planilha - Linhas coloridas por status - Ordenadas do menor para o maior % de atingimento SEÇÃO DE PRÓXIMOS PASSOS: - Os 3 Key Results mais críticos em cards destacados - 1 ação sugerida para cada um RODAPÉ: - Nome da empresa e data de geração Salve o arquivo como "dashboard-okrs-q1-2026.html" nesta pasta. O arquivo deve abrir direto no navegador sem precisar de internet.

Atividade Prática
08. Passo a passo
6 passos em até 12 minutos
1
Abra o Claude Desktop e acesse o Claude Code. (30 seg)
Abra o Claude Desktop. Na parte superior do app, clique na aba Code.
2
Selecione a pasta Delta Board. (30 seg)
Clique em "Select folder" ou "Selecionar pasta". Navegue até Documentos → Delta Board e confirme.
3
Cole e rode o prompt. (1 min)
No campo de texto do Claude Code, cole o prompt abaixo substituindo as variáveis entre [colchetes] pelos seus dados.
4
Acompanhe o Claude Code trabalhando. (5 a 7 min)
O Claude Code vai mostrar cada passo — lendo os arquivos, construindo o dashboard, aplicando as cores, salvando o HTML. É o Claude lendo, pensando e construindo em tempo real.
5
Abra o dashboard no navegador. (1 min)
Abra a pasta Delta Board. Você vai ver dashboard-okrs-[TRIMESTRE].html. Clique duas vezes — abre no navegador como um dashboard interativo.
6
Ajuste se necessário. (2 min)
Se algo não estiver certo, volte ao Claude Code e peça a alteração em linguagem natural. O Claude atualiza o arquivo e você recarrega o navegador para ver o resultado.
